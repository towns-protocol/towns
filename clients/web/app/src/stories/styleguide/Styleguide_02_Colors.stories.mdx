# Color System

## Background

The `background` property is semantic and will adapt the background-fill of a container
depending to the current theme. The text color will also adapt to the choosen background
unless its explicitely overriden by the `color` property.
The backgrounds are described in levels where the first one is also the default (think of the
ground-floor in a building).
If the `background` property is omitted, the container will remain transparent and
pass-through the background color of its parent container.

<h4>Example:</h4>

```tsx
<Stack background="level3">
  <Text>Default text color</Text>
</Stack>
```
